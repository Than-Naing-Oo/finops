Welcome to your new dbt project!

### Using the dbt starter project

Try running the following commands:
- dbt init 
- dbt debug 
- dbt run
- dbt test

### adding dependencies 
- add dependencies in project 
- create a package.yml as a same level of dbt_project.yml
